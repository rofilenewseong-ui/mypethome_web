04

개발 환경 세팅

5단계로 바로 시작하기

1

사전 준비

Node.js 18+, pnpm 9+ 필요
npm install -g pnpm
2

의존성 설치

git clone <repo-url> petholo && cd petholo && pnpm install
3

환경 변수 설정

cp .env.example apps/api/.env
Firebase 인증 정보, JWT 키, API 키 등 설정 (아래 섹션 참고)
4

Firebase 서비스 계정

cp /path/to/firebase-service-account.json apps/api/
Firebase Console에서 다운로드한 JSON 파일
5

실행!

pnpm dev
프론트 :3000 + 백엔드 :4000 동시 실행

10

트러블슈팅

자주 만나는 문제와 해결 방법

Firebase 연결 안됨

Firebase initialized in limited mode
➔ firebase-service-account.json이 apps/api/ 에 있는지 확인
➔ .env의 FIREBASE_* 변수 확인

CORS 에러

Access-Control-Allow-Origin 에러
➔ .env의 CORS_ORIGIN이 http://localhost:3000 인지 확인

API 호출 실패

Network Error / 데모 데이터 표시
➔ 정상 동작! 데모 폴백 패턴
➔ 실제 연동 필요 시 백엔드 실행 확인: localhost:4000/api/health

AI 기능 안됨

403 Forbidden / Silver tier required
➔ Silver 티어 이상만 AI 사용 가능
➔ API 키 환경변수 확인 (KLING_API_KEY 등)
