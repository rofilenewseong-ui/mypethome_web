## 버전1: heaven-box-mvp

> "일단 돌아가게 만든 첫 번째 시도"

**뭘로 만들었나?**
- 화면: HTML 파일을 직접 작성 (프레임워크 없음)
- 동작: JavaScript 파일로 버튼 클릭 등 처리
- 서버: Express (Node.js로 만든 간단한 서버)
- 데이터 저장: JSON 파일 (메모장 같은 텍스트 파일)

**화면이 2개다:**
- `index.html` → PC용 화면 (가로로 넓은 레이아웃)
- `mobile.html` → 모바일용 화면 (세로로 긴 레이아웃)

## 버전2: heaven-box-v1

> "React를 써서 좀 더 제대로 만든 버전"

**뭘로 만들었나?**
- 화면: Next.js + React (컴포넌트 방식)
- 스타일: Tailwind CSS (class 이름으로 디자인)
- 서버: 없음! (모든 데이터를 브라우저에 저장)
- 데이터 저장: localStorage (브라우저 내장 저장소)

- 서버가 없으니까 "가짜 로그인" 사용

## 버전3: petmorial

> "실제 출시를 목표로 만든 최종 버전"

**뭘로 만들었나?**
- 화면: React + TypeScript (타입 검사 추가)
- 서버: Express + SQLite (진짜 데이터베이스)
- AI연동: NanoBanana (이미지 생성) + Kling (영상 생성)
- 상태관리: Zustand (React에서 데이터 공유하는 도구)

**특징:**
- 화면이 15개나 된다 (로그인, 홈, 펫관리, 채팅 등)
- 관리자 페이지가 있다 (엑셀로 구매코드 등록)
- 실제 AI API를 연동해서 영상을 만들 수 있다
