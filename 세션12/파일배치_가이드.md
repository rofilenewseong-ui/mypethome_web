# 파일 배치 가이드

이 문서는 세션12의 파일들을 **프로젝트의 어디에 넣어야 하는지** 안내합니다.

> 프로젝트 루트: `petholo/apps/api/`

---

## 새 파일 (3개) - 새로 생성해야 하는 파일

| 파일 | 복사 위치 | 설명 |
|------|----------|------|
| `firestore-helpers.ts` | `src/utils/firestore-helpers.ts` | DB 쿼리를 효율적으로 묶어서 실행하는 헬퍼 |
| `sanitize.ts` | `src/utils/sanitize.ts` | 사용자 입력에서 HTML 태그 제거 (보안) |
| `migrate-denormalize.ts` | `database/migrate-denormalize.ts` | 기존 DB 데이터에 새 필드 추가하는 스크립트 |

---

## 파일 복사 명령어 (한번에 실행)

```bash
# 프로젝트 루트에서 실행 (petholo/apps/api/)
# 세션12 폴더 경로를 SESSION_DIR에 지정하세요

SESSION_DIR="세션12 폴더 경로"

# 새 파일
cp "$SESSION_DIR/새파일/firestore-helpers.ts" src/utils/
cp "$SESSION_DIR/새파일/sanitize.ts" src/utils/
cp "$SESSION_DIR/새파일/migrate-denormalize.ts" database/

# 수정 파일
cp "$SESSION_DIR/수정파일/services/"*.ts src/services/
cp "$SESSION_DIR/수정파일/controllers/"*.ts src/controllers/
cp "$SESSION_DIR/수정파일/routes/"*.ts src/routes/
cp "$SESSION_DIR/수정파일/schemas/"*.ts src/schemas/
cp "$SESSION_DIR/수정파일/types/"*.ts src/types/
cp "$SESSION_DIR/수정파일/docs/"*.md database/docs/
```
