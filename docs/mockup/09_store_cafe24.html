🔸 핵심 포인트: 코드 등록 없음
사용자는 어떤 코드도 입력할 필요가 없습니다. 카페24 구매 시 Webhook이 자동으로 PetHolo API로 발송되고, HMAC-SHA256 검증을 거친 후 member_id 매칭을 통해 해당 회원의 크레딧이 즉시 충전됩니다.

4. Webhook 연동 사양

카페24 자사몰 주문 완료 시 Webhook을 통한 자동 충전 기술 사양입니다.

Webhook 사양

EndpointPOST /api/webhook/cafe24/order
인증HMAC-SHA256 (X-Cafe24-Hmac-SHA256)
Payloadorder_id, member_id, product_code, amount
응답200 OK + credit_charged confirmation
재시도3회 (5분 간격)
POST /api/webhook/cafe24/order JSON 
{
  "event": "order.completed",
  "order_id": "20260305-001",
  "member_id": "usr_a1b2c3d4e5",
  "product_code": "PETHOLO-CREDIT-60",
  "amount": 45000,
  "credit_amount": 60,
  "paid_at": "2026-03-05T14:30:00+09:00"
}

Security: HMAC-SHA256 검증

🔒 Webhook 보안 검증
  • ✓HMAC-SHA256 서명 검증으로 요청 무결성 확인
  • ✓X-Cafe24-Hmac-SHA256 헤더에 서명 값 포함
  • ✓공유 비밀키(Shared Secret)로 서명 생성 및 검증
  • ✓Timestamp 검증으로 재전송 공격(Replay Attack) 방지
  • ✓IP 허용 목록(Allowlist)으로 Cafe24 서버만 허용
200 OK Response JSON

5. API 엔드포인트

카페24 연동 및 크레딧 관련 API 엔드포인트 목록입니다.

POST /api/webhook/cafe24/order
Cafe24 주문 완료 Webhook 수신. HMAC-SHA256 서명 검증 후 member_id 매칭 및 크레딧 자동 충전.
GET /api/credits
사용자 크레딧 잔액 조회. 현재 보유 크레딧 및 등급 정보 반환.
GET /api/credits/history
크레딧 충전/사용 내역 조회. 자동충전, 사용, 환불 타입 필터링 및 페이지네이션 지원.
