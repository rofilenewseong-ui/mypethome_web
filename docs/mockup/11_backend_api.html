PetHolo Backend API Specification

Express.js 5 + Firebase Firestore 기반 REST API 전체 명세서
Base URL: /api | Auth: JWT (Cafe24 SSO) | Version: 1.0

1 Auth - Cafe24 SSO 2 Pets - 반려동물 3 Profiles - 프로필 4 Base Videos - 베이스 영상 5 Motions - 모션 6 AI - 생성 파이프라인 7 Messenger - AI 메신저 8 Credits - 크레딧 9 Trash - 휴지통 10 Webhooks - Cafe24 11 Admin - 관리자 * Auth Flow Diagram * Error Format * Rate Limiting * Firestore Schema * Middleware Pipeline

Tech Stack

  • Express.js 5 (ESM)
  • Firebase Firestore (NoSQL)
  • Firebase Storage (Media)
  • JWT (Access + Refresh Token)
  • Zod (Request Validation)
  • Helmet + CORS + Rate Limit

Authentication

  • Cafe24 SSO 토큰 교환 방식
  • Google OAuth 단독 사용 안함
  • Access Token: 1시간 만료
  • Refresh Token: 30일 만료
  • Header: Authorization: Bearer {token}
  • Admin: role 기반 접근 제어

Credit System

  • Cafe24 주문 웹훅으로 자동 충전
  • 수동 코드 등록 없음 (NO redeem-code)
  • 베이스 영상 생성: 40C
  • 모션 생성: 40C
  • AI 이미지 재생성: 10C
  • 삭제 환불: 20C (영상/모션)

Conventions

  • RESTful JSON API
  • UTC timestamps (ISO 8601)
  • Pagination: cursor 기반
  • Error: { code, message, details }
  • Soft delete 기본 (휴지통)
  • HMAC-SHA256 웹훅 검증

1 Auth - Cafe24 SSO 인증

Cafe24 쇼핑몰에서 진입한 사용자의 SSO 토큰을 교환하여 JWT를 발급합니다. Google OAuth 단독 로그인은 사용하지 않습니다.
Base Path: /api/auth

POST /api/auth/cafe24

Cafe24 SSO 토큰을 PetHolo JWT로 교환합니다. 신규 사용자인 경우 자동으로 계정이 생성됩니다. Cafe24 상점에서 리다이렉트 시 전달받은 authorization code를 사용합니다.

Request Body
{ "cafe24Code": "auth_code_from_cafe24_redirect", "mallId": "petholo-shop", "redirectUri": "https://app.petholo.com/auth/callback" }
Response 200
{ "accessToken": "eyJhbGciOiJIUzI1NiIs...", "refreshToken": "dGhpcyBpcyByZWZyZXNo...", "expiresIn": 3600, "user": { "uid": "usr_abc123", "cafe24MemberId": "member_xyz", "email": "user@example.com", "displayName": "홍길동", "tier": "silver", "credits": 200, "isNew": false, "createdAt": "2026-01-15T09:00:00Z" } }
POST /api/auth/refresh
AUTH

Refresh Token을 사용하여 새로운 Access Token을 발급합니다. Refresh Token은 1회 사용 후 교체됩니다 (rotation).

Request Body
{ "refreshToken": "dGhpcyBpcyByZWZyZXNo..." }
Response 200
{ "accessToken": "eyJhbGciOiJIUzI1NiIs...", "refreshToken": "bmV3IHJlZnJlc2ggdG9r...", "expiresIn": 3600 }
POST /api/auth/logout
AUTH

현재 세션을 종료하고 Refresh Token을 무효화합니다. FCM 토큰도 함께 제거됩니다.

Request Body
{ "refreshToken": "dGhpcyBpcyByZWZyZXNo..." }
Response 200
{ "message": "로그아웃되었습니다." }
GET /api/auth/me
AUTH

현재 로그인된 사용자의 정보를 반환합니다. 크레딧 잔액, 등급, 펫 수 등 요약 정보를 포함합니다.

Response 200
{ "uid": "usr_abc123", "cafe24MemberId": "member_xyz", "email": "user@example.com", "displayName": "홍길동", "tier": "silver", "credits": 200, "petCount": 2, "profileCount": 3, "fcmToken": "fcm_token_here", "createdAt": "2026-01-15T09:00:00Z", "lastLoginAt": "2026-03-05T14:30:00Z" }

2 Pets - 반려동물 관리

반려동물 CRUD. 각 사용자는 최대 5마리의 반려동물을 등록할 수 있습니다. 삭제 시 휴지통으로 이동(soft delete)합니다.
Base Path: /api/pets

POST /api/pets
AUTH

새 반려동물을 등록합니다. 기본 정보와 사진(정면/전신)을 입력합니다.

Request Body
{ "name": "보리", "species": "dog", "breed": "골든 리트리버", "birthDate": "2018-03-15", "passedDate": "2025-12-01", "frontPhoto": "https://storage.../front.jpg", "sidePhoto": "https://storage.../side.jpg", "favoriteSnack": "닭가슴살", "walkingPlace": "한강공원" }
Response 201
{ "id": "pet_a1b2c3", "name": "보리", "species": "dog", "breed": "골든 리트리버", "birthDate": "2018-03-15", "passedDate": "2025-12-01", "frontPhoto": "https://storage.../front.jpg", "sidePhoto": "https://storage.../side.jpg", "favoriteSnack": "닭가슴살", "walkingPlace": "한강공원", "profileCount": 0, "ownerId": "usr_abc123", "createdAt": "2026-03-05T10:00:00Z" }
GET /api/pets
AUTH

현재 사용자의 모든 반려동물 목록을 조회합니다. 프로필 수 요약을 포함합니다.

Response 200
{ "pets": [ { "id": "pet_a1b2c3", "name": "보리", "species": "dog", "frontPhoto": "https://...", "profileCount": 2 }, { "id": "pet_d4e5f6", "name": "나비", "species": "cat", "frontPhoto": "https://...", "profileCount": 1 } ], "total": 2 }
GET /api/pets/:petId
AUTH

특정 반려동물의 상세 정보를 조회합니다. 프로필 목록과 함께 반환됩니다.

Response 200
{ "id": "pet_a1b2c3", "name": "보리", "species": "dog", "breed": "골든 리트리버", "birthDate": "2018-03-15", "passedDate": "2025-12-01", "frontPhoto": "https://storage.../front.jpg", "sidePhoto": "https://storage.../side.jpg", "favoriteSnack": "닭가슴살", "walkingPlace": "한강공원", "profiles": [ { "id": "prf_001", "name": "보리 프로필 1", "status": "active" } ], "ownerId": "usr_abc123", "createdAt": "2026-03-05T10:00:00Z", "updatedAt": "2026-03-05T12:00:00Z" }
PUT /api/pets/:petId
AUTH

반려동물 정보를 수정합니다. 부분 수정 지원 (전송된 필드만 업데이트).

Request Body
{ "name": "보리 (수정)", "personality": "활발하고 산책을 좋아함" }
Response 200
{ "id": "pet_a1b2c3", "name": "보리 (수정)", "personality": "활발하고 산책을 좋아함", "updatedAt": "2026-03-05T15:00:00Z" }
DELETE /api/pets/:petId
AUTH

반려동물을 휴지통으로 이동합니다 (soft delete). 관련 프로필, 영상, 모션도 함께 이동됩니다. 30일 후 자동 영구 삭제.

Response 200
{ "message": "휴지통으로 이동되었습니다.", "trashedAt": "2026-03-05T15:00:00Z", "permanentDeleteAt": "2026-04-04T15:00:00Z" }

3 Profiles - 프로필 관리

반려동물별 홀로그램 프로필. 각 프로필에는 베이스 영상(3슬롯)과 모션(12슬롯)이 포함됩니다.
Base Path: /api/profiles

POST /api/profiles
AUTH

새 홀로그램 프로필을 생성합니다. 반드시 petId를 지정해야 합니다.

Request Body
{ "petId": "pet_a1b2c3", "name": "보리 봄 프로필", "description": "봄 테마 홀로그램" }
Response 201
{ "id": "prf_001", "petId": "pet_a1b2c3", "name": "보리 봄 프로필", "description": "봄 테마 홀로그램", "baseVideoSlots": { "max": 3, "used": 0 }, "motionSlots": { "max": 12, "used": 0 }, "status": "active", "createdAt": "2026-03-05T10:00:00Z" }
GET /api/profiles?petId=pet_a1b2c3
AUTH

특정 반려동물의 프로필 목록을 조회합니다. petId 쿼리 파라미터 필수.

Response 200
{ "profiles": [ { "id": "prf_001", "name": "보리 봄 프로필", "baseVideoSlots": { "max": 3, "used": 2 }, "motionSlots": { "max": 12, "used": 5 }, "status": "active" } ], "total": 1 }
GET /api/profiles/:profileId
AUTH

프로필 상세 조회. 베이스 영상과 모션 슬롯 현황을 포함합니다.

Response 200
{ "id": "prf_001", "petId": "pet_a1b2c3", "name": "보리 봄 프로필", "description": "봄 테마 홀로그램", "gifUrl": "https://storage.../gifs/base-videos/bv001.gif", // 활성 베이스 영상의 gifUrl 파생 "baseVideos": [ { "slotIndex": 0, "id": "bv_001", "status": "active", "thumbnailUrl": "...", "gifUrl": "..." }, { "slotIndex": 1, "id": "bv_002", "status": "generating", "thumbnailUrl": null, "gifUrl": null }, { "slotIndex": 2, "id": null, "status": "empty" } ], "motions": [ { "slotIndex": 0, "id": "mot_001", "label": "앉기", "status": "active", "gifUrl": null } ], "status": "active", "createdAt": "2026-03-05T10:00:00Z" }
PUT /api/profiles/:profileId
AUTH

프로필 이름/설명을 수정합니다.

Request Body
{ "name": "보리 여름 프로필", "description": "여름 테마로 변경" }
Response 200
{ "id": "prf_001", "name": "보리 여름 프로필", "description": "여름 테마로 변경", "updatedAt": "2026-03-05T16:00:00Z" }
DELETE /api/profiles/:profileId
AUTH

프로필을 휴지통으로 이동합니다. 관련 베이스 영상과 모션도 함께 이동됩니다.

Response 200
{ "message": "프로필이 휴지통으로 이동되었습니다.", "trashedAt": "2026-03-05T16:00:00Z", "permanentDeleteAt": "2026-04-04T16:00:00Z" }

4 Base Videos - 베이스 영상

홀로그램의 기본 영상(정면 정지 포즈). 프로필당 최대 3슬롯. 생성 시 40크레딧 소모, 삭제 시 20크레딧 환불.
Base Path: /api/base-videos

POST /api/base-videos
AUTH -40C

새 베이스 영상 생성을 요청합니다. AI 파이프라인(NanoBanana 이미지 생성 -> Kling 비디오 생성)이 시작됩니다. 40크레딧이 즉시 차감됩니다.

Request Body
{ "profileId": "prf_001", "slotIndex": 0, "sourcePhotoUrl": "https://storage.../pet_photo.jpg", "style": "realistic", "pose": "standing_front" }
Response 201
{ "id": "bv_003", "profileId": "prf_001", "slotIndex": 0, "status": "generating_image", "aiJobId": "job_img_abc123", "creditsCharged": 40, "remainingCredits": 160, "estimatedMinutes": 8, "createdAt": "2026-03-05T10:00:00Z" }
GET /api/base-videos?profileId=prf_001
AUTH

프로필에 속한 베이스 영상 목록을 조회합니다. 슬롯 인덱스 순으로 정렬됩니다.

Response 200
{ "baseVideos": [ { "id": "bv_001", "slotIndex": 0, "status": "active", "thumbnailUrl": "https://storage.../thumb_bv001.jpg", "videoUrl": "https://storage.../bv001.mp4", "gifUrl": "https://storage.../gifs/base-videos/bv001.gif", "isActive": true }, { "id": "bv_002", "slotIndex": 1, "status": "generating_video", "progress": 65, "gifUrl": null, "isActive": false } ], "total": 2, "maxSlots": 3 }
GET /api/base-videos/:baseVideoId
AUTH

베이스 영상의 상세 정보 및 생성 진행 상태를 조회합니다.

Response 200
{ "id": "bv_001", "profileId": "prf_001", "slotIndex": 0, "status": "active", "thumbnailUrl": "https://storage.../thumb.jpg", "videoUrl": "https://storage.../video.mp4", "gifUrl": "https://storage.../gifs/base-videos/bv001.gif", "sourcePhotoUrl": "https://storage.../photo.jpg", "aiGeneratedImageUrl": "https://storage.../ai_img.png", "isActive": true, "duration": 5.0, "resolution": "1024x1024", "createdAt": "2026-03-05T10:00:00Z" }
PATCH /api/base-videos/:baseVideoId/activate
AUTH

해당 슬롯의 베이스 영상을 활성화합니다. 같은 슬롯의 다른 영상은 비활성화됩니다.

Response 200
{ "id": "bv_001", "isActive": true, "message": "베이스 영상이 활성화되었습니다." }
DELETE /api/base-videos/:baseVideoId
AUTH +20C 환불

베이스 영상을 삭제합니다. 20크레딧이 환불됩니다. 휴지통으로 이동(soft delete).

Response 200
{ "message": "베이스 영상이 삭제되었습니다.", "refundedCredits": 20, "remainingCredits": 180, "trashedAt": "2026-03-05T15:00:00Z" }

5 Motions - 모션

홀로그램 모션 애니메이션. 프로필당 최대 12슬롯. 생성 시 40크레딧, 삭제 시 20크레딧 환불. 위치(position) 조정 가능.
Base Path: /api/motions

POST /api/motions
AUTH -40C

새 모션을 생성합니다. AI가 반려동물 이미지를 기반으로 동작 영상을 생성합니다. 40크레딧 즉시 차감.

Request Body
{ "profileId": "prf_001", "slotIndex": 0, "motionType": "sit", "label": "앉기", "sourceImageUrl": "https://storage.../ai_pet.png", "prompt": "golden retriever sitting down gently" }
Response 201
{ "id": "mot_001", "profileId": "prf_001", "slotIndex": 0, "label": "앉기", "motionType": "sit", "status": "generating", "aiJobId": "job_vid_def456", "creditsCharged": 40, "remainingCredits": 120, "estimatedMinutes": 5, "createdAt": "2026-03-05T10:00:00Z" }
GET /api/motions?profileId=prf_001
AUTH

프로필의 모션 목록을 조회합니다. 슬롯 인덱스 순으로 정렬됩니다.

Response 200
{ "motions": [ { "id": "mot_001", "slotIndex": 0, "label": "앉기", "motionType": "sit", "status": "active", "thumbnailUrl": "https://storage.../thumb_mot001.jpg", "position": { "x": 0, "y": 0, "scale": 1.0 } } ], "total": 1, "maxSlots": 12 }
GET /api/motions/:motionId
AUTH

모션 상세 정보를 조회합니다. 영상 URL, 위치 정보, 생성 상태 등을 포함.

Response 200
{ "id": "mot_001", "profileId": "prf_001", "slotIndex": 0, "label": "앉기", "motionType": "sit", "status": "active", "videoUrl": "https://storage.../mot001.mp4", "thumbnailUrl": "https://storage.../thumb.jpg", "duration": 3.0, "position": { "x": 0, "y": 0, "scale": 1.0 }, "createdAt": "2026-03-05T10:00:00Z" }
PATCH /api/motions/:motionId/position
AUTH

모션의 홀로그램 프리즘 내 위치(x, y)와 크기(scale)를 조정합니다.

Request Body
{ "x": 10, "y": -5, "scale": 1.2 }
Response 200
{ "id": "mot_001", "position": { "x": 10, "y": -5, "scale": 1.2 }, "message": "위치가 업데이트되었습니다." }
DELETE /api/motions/:motionId
AUTH +20C 환불

모션을 삭제합니다. 20크레딧 환불. 휴지통으로 이동(soft delete).

Response 200
{ "message": "모션이 삭제되었습니다.", "refundedCredits": 20, "remainingCredits": 140, "trashedAt": "2026-03-05T15:00:00Z" }

6 AI - 생성 파이프라인

Gemini(스타트 프레임 이미지 생성) + Kling(비디오 생성) + ffmpeg(GIF 변환) AI 파이프라인을 관리합니다. 이미지 선택 제한시간: 30분.
이미지 입력: URL(펫 등록 사진)과 base64(의상 사진) 혼합 지원 — 백엔드에서 URL→base64 자동 변환.
Base Path: /api/ai

POST /api/ai/generate-image
AUTH

NanoBanana AI를 통해 반려동물 사진 기반 이미지를 생성합니다. 4장의 후보 이미지가 생성됩니다. 베이스 영상/모션 생성 플로우의 첫 단계.

Request Body
{ "sourcePhotoUrl": "https://storage.../pet_photo.jpg", "style": "realistic", "pose": "standing_front", "petId": "pet_a1b2c3", "targetType": "base_video" }
Response 202
{ "jobId": "job_img_abc123", "status": "processing", "estimatedSeconds": 30, "message": "이미지 생성이 시작되었습니다." }
POST /api/ai/select-image
AUTH

생성된 4장의 후보 이미지 중 하나를 선택합니다. 선택 제한시간 30분 (초과 시 job 만료). 선택 후 비디오 생성이 자동으로 시작됩니다.

Request Body
{ "jobId": "job_img_abc123", "selectedIndex": 2 // 0-3 중 선택 }
Response 200
{ "jobId": "job_img_abc123", "selectedImageUrl": "https://storage.../selected.png", "videoJobId": "job_vid_def456", "status": "video_generating", "message": "이미지 선택 완료. 비디오 생성이 시작됩니다." }
POST /api/ai/regenerate
AUTH -10C

현재 이미지 후보가 마음에 들지 않을 때 새로 생성합니다. 10크레딧 추가 차감. 기존 job의 이미지를 교체합니다.

Request Body
{ "jobId": "job_img_abc123", "reason": "포즈가 다릅니다" // 선택사항 }
Response 202
{ "jobId": "job_img_abc123", "status": "regenerating", "creditsCharged": 10, "remainingCredits": 110, "regenerateCount": 1, "message": "이미지를 다시 생성합니다." }
GET /api/ai/jobs/:jobId
AUTH

AI 작업의 현재 상태를 폴링합니다. 프론트엔드에서 주기적으로 호출하여 진행률을 확인합니다.

Response 200 - 이미지 생성 완료
{ "jobId": "job_img_abc123", "type": "image_generation", "status": "completed", "progress": 100, "candidates": [ { "index": 0, "url": "https://storage.../cand_0.png" }, { "index": 1, "url": "https://storage.../cand_1.png" }, { "index": 2, "url": "https://storage.../cand_2.png" }, { "index": 3, "url": "https://storage.../cand_3.png" } ], "expiresAt": "2026-03-05T10:30:00Z", "createdAt": "2026-03-05T10:00:00Z" }
Response 200 - 비디오 생성 진행 중
{ "jobId": "job_vid_def456", "type": "video_generation", "status": "processing", "progress": 65, "estimatedRemainingSeconds": 120, "createdAt": "2026-03-05T10:01:00Z" }
POST /api/ai/generate-video
AUTH

Kling AI를 통해 선택된 이미지를 비디오로 변환합니다. 일반적으로 select-image에서 자동 호출되지만 수동 트리거도 가능합니다.

Auth Flow Diagram

Cafe24 SSO -> Token Exchange -> JWT 발급 인증 플로우

1. 사용자 로그인 (Cafe24 SSO)

Cafe24 Shop -> 사용자 로그인 클릭 -> Cafe24 OAuth 페이지 -> Authorization Code 발급

2. 토큰 교환

Frontend Callback -> POST /api/auth/cafe24 -> Cafe24 Token API -> 회원 정보 조회

3. JWT 발급 & 세션 생성

사용자 조회/생성 -> Firestore users -> JWT 생성 (Access+Refresh) -> 토큰 저장 (httpOnly cookie)

4. API 요청 인증

API 요청 + Bearer Token -> JWT 검증 미들웨어 -> req.user 설정 -> 라우터 핸들러 실행

5. 크레딧 자동 충전 (웹훅)

Cafe24 주문 완료 -> Webhook POST -> HMAC 검증 -> 크레딧 자동 충전 -> FCM 알림

Middleware Pipeline

Express.js 요청 처리 파이프라인. 모든 요청은 아래 순서대로 미들웨어를 통과합니다.

Request In -> Helmet (Security Headers) -> CORS -> Rate Limiter
-> Body Parser (JSON) -> Request Logger (Morgan) -> Request ID (uuid)
-> Auth Middleware (JWT) -> Role Check (Admin) -> Zod Validation
-> Route Handler -> Error Handler -> Response Out
MiddlewarePackageDescription (Korean)
helmet()helmet보안 HTTP 헤더 설정 (XSS, MIME sniffing 방지)
cors()corsCross-Origin 요청 허용 (허용 도메인 제한)
rateLimit()express-rate-limitIP/사용자 기반 요청 속도 제한
express.json()expressJSON 요청 본문 파싱 (10mb 제한)
morgan()morganHTTP 요청 로깅 (combined format)
requestId()uuid요청별 고유 ID 생성 (추적 용도)
authenticate()customJWT 검증 및 req.user 설정
requireAdmin()customrole="admin" 확인 (관리자 전용)
validate(schema)zodZod 스키마 기반 요청 유효성 검사
verifyHmac()customCafe24 웹훅 HMAC-SHA256 서명 검증
errorHandler()custom통합 에러 핸들러 (에러 포맷 변환 & 로깅)
