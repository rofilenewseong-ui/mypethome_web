3. Specifications

크레딧 비용, 등급 비교, 환불 정책 상세 사양입니다.

3-1. 크레딧 비용 분류 (Credit Cost Breakdown)

작업 (Action) 크레딧 비용 설명 비고
베이스 영상 생성 40C 사진 기반 3D 홀로그램 베이스 영상 생성 AI 모델 추론 비용 포함
모션 생성 40C 베이스 영상에 모션(움직임) 적용 베이스 영상 필수 선행
이미지 재생성 10C 기존 생성 결과 재시도 (다른 결과) 동일 소스 이미지 기반
삭제 시 환불 +50% 생성물 삭제 시 원래 비용의 50% 환불 40C 생성물 삭제 = +20C 환불

3-2. 등급 비교표 (Tier Comparison)

항목 Bronze (체험) Silver (정식)
가입 조건 카페24 SSO 로그인 (무료) 카페24에서 풀세트 구매
초기 크레딧 0C 120C
베이스 영상 생성 불가 가능 (40C)
모션 생성 불가 가능 (40C)
이미지 재생성 불가 가능 (10C)
홀로그램 재생 불가 가능
크레딧 추가 구매 불가 가능
앱 탐색 / UI 조회 가능 가능

3-3. 환불 정책 (Refund Rules)

규칙 상세
삭제 환불 비율 원래 생성 비용의 50% 자동 환불
베이스 영상 삭제 40C 소비 → 삭제 시 +20C 환불
모션 삭제 40C 소비 → 삭제 시 +20C 환불
이미지 재생성 삭제 10C 소비 → 삭제 시 +5C 환불
환불 시점 삭제 확인 즉시 (동기 처리)
환불 기록 거래 내역에 환불 트랜잭션 자동 기록 (immutable)
크레딧 유효기간 무제한 (만료 없음)

3-4. 구매 / 통합 정보 (Purchase & Integration)

항목 상세
결제 연동 Cafe24 쇼핑몰 PG 결제 시스템 연동
카페24 자동충전 방식 카페24 자사몰에서 풀세트 구매 → 주문 연동 → 크레딧 자동충전 + 등급 업그레이드
40 크레딧 팩 ₩12,000 (크레딧당 ₩300)
120 크레딧 팩 ₩29,000 (크레딧당 ₩242, 19% 할인)
거래 내역 특성 Immutable audit trail — 모든 크레딧 변동은 불변 기록으로 저장, 수정/삭제 불가
등급 업그레이드 방법 카페24에서 풀세트 구매만 가능 (앱 내 직접 업그레이드 없음)
