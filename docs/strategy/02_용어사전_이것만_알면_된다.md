# 이것만 알면 코드가 보인다 - 용어 사전
> 출력해서 모를때마다 찾아보세요!

---

## 파일 종류

| 파일 끝이 | 뭐하는 파일? | 비유 |
|-----------|-------------|------|
| .html | 화면의 뼈대 (글자, 버튼, 입력칸 배치) | 건물 설계도 |
| .css | 화면의 색깔, 크기, 간격 (예쁘게 꾸미기) | 인테리어 |
| .js | 버튼 누르면 뭐가 실행되는지 (동작) | 전기 배선 |
| .ts / .tsx | .js랑 같은데 "타입"이라는 안전장치 추가 | 전기 배선 + 누전차단기 |
| .json | 설정 파일이나 데이터 저장 | 메모장 |
| .env | 비밀번호, API 키 같은 비밀 정보 | 금고 |
| .db | 데이터베이스 파일 (정보 저장) | 서류 캐비넷 |

## 코드에서 자주 보는 단어

### 데이터 관련

| 단어 | 뜻 | 쉽게 |
|------|-----|------|
| state (상태) | 현재 앱이 기억하고 있는 정보 | "지금 로그인 했나? 크레딧 몇개?" |
| variable (변수) | 값을 담는 상자 | 이름표 붙은 상자 |
| const | 한번 넣으면 못 바꾸는 상자 | 잠긴 상자 |
| let | 바꿀 수 있는 상자 | 열린 상자 |
| array (배열) | 여러 개를 순서대로 담는 상자 | 서랍장 |
| object (객체) | 이름표가 붙은 여러 값의 묶음 | 프로필 카드 |
| localStorage | 브라우저가 기억하는 저장소 | 폰의 메모앱 |
| database (DB) | 서버가 기억하는 저장소 | 회사 서류 캐비넷 |

### 화면 관련

| 단어 | 뜻 | 쉽게 |
|------|-----|------|
| DOM | 화면에 보이는 모든 요소들 | 화면에 있는 버튼, 글자, 사진 전부 |
| element (요소) | 화면의 한 조각 | 버튼 1개, 글자 1줄 |
| render (렌더) | 화면에 그려주기 | 화면 새로고침 |
| component (컴포넌트) | 재사용 가능한 화면 조각 | 레고 블록 |
| view (뷰) | 화면 1장 | 앱의 한 페이지 |
| route (라우트) | URL 주소별로 어떤 화면 보여줄지 | 네비게이션 |
| modal (모달) | 화면 위에 뜨는 팝업 | 알림창 |

### 서버 관련

| 단어 | 뜻 | 쉽게 |
|------|-----|------|
| server (서버) | 데이터를 보관하고 처리하는 컴퓨터 | 식당 주방 |
| client (클라이언트) | 유저가 보는 화면 | 식당 홀 |
| API | 서버와 화면이 대화하는 방법 | 주문서 |
| endpoint (엔드포인트) | API의 주소 하나하나 | 주문 메뉴 1개 |
| request (요청) | 화면 → 서버로 보내는 요청 | "이거 주세요" |
| response (응답) | 서버 → 화면으로 보내는 답변 | "여기 있습니다" |
| GET | 데이터 달라고 요청 | "메뉴판 보여주세요" |
| POST | 데이터를 보내는 요청 | "이거 주문할게요" |
| token (토큰) | 로그인 했다는 증명서 | 출입증 |
| JWT | 토큰의 한 종류 | 디지털 출입증 |
| middleware | 요청이 들어올 때 먼저 확인하는 관문 | 경비실 |

### 함수 관련

| 단어 | 뜻 | 쉽게 |
|------|-----|------|
| function (함수) | "이거 하세요" 라는 명령 묶음 | 레시피 |
| async / await | 시간 걸리는 작업을 기다림 | "다 될 때까지 기다려" |
| callback | 뭔가 끝나면 실행되는 함수 | "다 되면 알려줘" |
| event listener | 클릭/입력 등을 감시하다가 반응 | "버튼 눌리면 이거 해" |
| setInterval | 일정 시간마다 반복 실행 | 알람시계 |
| setTimeout | 일정 시간 후 1번 실행 | 타이머 |

### React 관련

| 단어 | 뜻 | 쉽게 |
|------|-----|------|
| useState | 컴포넌트가 기억하는 값 | "이 화면은 이걸 기억해" |
| useEffect | 화면이 열릴 때 자동 실행 | "화면 열리면 이거 해" |
| props | 부모가 자식에게 전달하는 값 | "이 정보 갖고 가서 보여줘" |
| Zustand | 여러 화면이 같은 정보를 공유 | 공용 게시판 |

## 자주 나오는 약어

| 약어 | 풀네임 | 뜻 |
|------|--------|-----|
| MVP | Minimum Viable Product | 최소한으로 동작하는 첫 번째 버전 |
| UI | User Interface | 유저가 보는 화면 |
| DB | Database | 데이터 저장소 |
| API | Application Programming Interface | 프로그램끼리 대화하는 방법 |
| JWT | JSON Web Token | 디지털 출입증 |
| CSS | Cascading Style Sheets | 화면 꾸미기 언어 |
| HTML | HyperText Markup Language | 화면 뼈대 언어 |
| JS | JavaScript | 동작 처리 언어 |
| TS | TypeScript | JS + 안전장치 |
| URL | Uniform Resource Locator | 웹 주소 |
| CRUD | Create Read Update Delete | 만들기/보기/수정/삭제 |
| SPA | Single Page Application | 페이지 이동 없이 화면만 바뀌는 앱 |
