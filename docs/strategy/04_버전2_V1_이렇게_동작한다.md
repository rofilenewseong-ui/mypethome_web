## 버전1과 뭐가 다른가?

| 항목 | 버전1 (MVP) | 버전2 (V1) |
|------|-----------|-----------|
| 만든 방식 | HTML 직접 작성 | React (레고 블록 조립) |
| 서버 | 있음 (Express) | 없음! (전부 브라우저에서 처리) |
| 로그인 | 진짜 Google 로그인 | 가짜 로그인 (이메일만 입력) |
| 데이터 저장 | 서버 JSON 파일 | 브라우저 localStorage |
| 크레딧 | 120 포인트 | 3회 (제작권) |
| 채팅 | 있음 | "준비 중" |
| 펫 정보 | 이름+종류 | 이름+생일+기일+견종+장례+거주지 (7가지) |

## 탭1: 영상 만들기 (HomeView)

### 로그인 안 한 상태
```
"로그인 후 이용 가능합니다."
```

### 로그인 했지만 펫 미등록
```
"아직 등록된 아이가 없어요."
[+ 우리 아이 등록하기]  ← 누르면 마이페이지 탭으로 이동
```

### 펫 등록 완료 상태
```
등록된 아이
┌──────────────────────┐
│ [사진]  뽀삐            │ ← 누르면 사진 업로드 팝업
│         클릭해서 사진 업로드│
└──────────────────────┘

최근 작업 상태: 완료
[홀로그램 플레이어 열기]  ← 전체화면 플레이어
```

## 탭2: 마이페이지 (MyPageView) - 동물 등록 폼

```
동물 등록 폼
┌──────────────────────┐
│ [강아지 사진 선택]       │ ← 필수!
│ [강아지 이름]            │ ← 필수
│ [생년월일] [기일]         │ ← 둘 다 필수
│ [견종]                  │ ← 필수
│ [장례 방식]              │ ← 필수
│ [장례 장소]              │ ← 필수
│ [거주지]                │ ← 필수
│ [동물 정보 저장]          │
└──────────────────────┘
```

**저장 누르면 일어나는 일:**
```
7개 필드 + 사진 전부 입력했나 확인
  → 하나라도 빈칸 → "모든 항목을 입력해주세요"
  → 사진 없음 → "강아지 프로필 사진을 등록해주세요"
  → 전부 입력함 → 브라우저 저장소에 펫 정보 저장!
```

## 사진 업로드 팝업 (UploadModal)

```
┌──────────────────────┐
│ 사진 3~4장 업로드        │
│ [파일 선택]              │
│ 선택: 3장                │
│ [취소]  [생성 시작]       │
└──────────────────────┘
```

**"생성 시작" 누르면:**
```
1. 사진이 3~4장인지 확인
2. 제작권이 남아있는지 확인 (3→2→1→0)
3. 제작권 1개 차감
4. "AI 영상 생성 시작" 토스트 표시
5. 2.4초 기다림 (실제로는 AI가 만드는 시간)
6. 영상 3종 완료!
   → idle.mp4 (기본 A)
   → pet.mp4  (쓰다듬기 B)
   → feed.mp4 (간식주기 C)
```

## 홀로그램 플레이어 (HologramPlayer)

```
전체 화면 검은 배경

┌──────────────────────────────┐
│                              │
│  [좌측 터치]  [영상]  [우측 터치]  │
│              82vw            │
│                              │
│                       [닫기]  │
└──────────────────────────────┘
```

**동작 방식:**
```
화면 열리면 → 기본(A) 영상 자동 재생 (반복)
좌측 터치   → 쓰다듬기(B) 영상 재생
우측 터치   → 간식주기(C) 영상 재생
3.2초 후    → 자동으로 기본(A)으로 돌아감

특별 기능: 화면 꺼짐 방지 (WakeLock)
→ 홀로그램 보는 동안 핸드폰 화면이 안 꺼짐!
```
